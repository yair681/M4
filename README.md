## 🚀 התקנה מהירה

### שלב 1: הורד את הפרויקט
```bash
# אם הורדת ZIP - פתח אותו
# אם משתמש ב-Git:
git clone <repository-url>
cd MasterCode_Manager
```

### שלב 2: התקן תלויות
**דרך 1: הרצה אוטומטית (Windows)**
```cmd
INSTALL.bat
```

**דרך 2: התקנה ידנית**
```bash
npm install
```

### שלב 3: הפעל את השרת
**דרך 1: הרצה אוטומטית (Windows)**
```cmd
start.bat
```

**דרך 2: הרצה ידנית**
```bash
npm start
```

### שלב 4: פתח דפדפן
עבור ל: `http://localhost:3000`

## 📖 מדריך שימוש

### ניהול פרויקטים וקבצים

1. **צור פרויקט חדש**
   - לחץ על "פרויקטים" בתפריט
   - לחץ "+ הוסף פרויקט"
   - מלא פרטים ושמור

2. **העלה קבצים**
   - גרור קבצים לאיזור ההעלאה
   - או לחץ לבחירת קבצים
   - הקבצים יישמרו אוטומטית

3. **ערוך קבצי קוד**
   - לחץ "ערוך" על כרטיס הקובץ
   - עורך הקוד ייפתח אוטומטית
   - ערוך ושמור עם Ctrl+S

4. **הרץ HTML**
   - לחץ "הרץ" בעורך (רק לקבצי HTML)
   - הקוד ירוץ בחלון נפרד

## 🔧 פתרון בעיות

### השרת לא עולה
```bash
# בדוק שהתלויות מותקנות
npm install

# נסה להריץ ידנית
node server.js
```

### עורך הקוד לא נטען
- בדוק חיבור אינטרנט (Monaco Editor נטען מ-CDN)
- פתח Console (F12) וחפש שגיאות
- נסה לרענן את הדף

### קבצים לא נשמרים
- וודא שלשרת יש הרשאות כתיבה לתיקייה
- בדוק שה-API עובד: `http://localhost:3000/api/data`

### AI לא עובד
- וודא ש-endpoint של `/api/ai-chat` עובד
- בדוק Console לשגיאות
